# mt-vue

> 基于 vue，仿美团外卖

## 线上地址

http://lococo.site/shop.html#/

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```
